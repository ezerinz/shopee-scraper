## Setup
- Install Google Chrome
- Clone github
```bash
git clone https://github.com/ezerinz/shopee-scraper
```
- Ganti direktori
```bash
cd shopee-scraper
```
- Install dependencies
```bash
pip install -r requirements.txt
```
- Jalankan server
```bash
gunicorn main:app
```
